Raspberry Pi AI Vision Desk Assistant
Hardware Requirements

Required:
- Raspberry Pi 5 8GB
- Official Raspberry Pi USB-C power supply (recommended 5V/5A)
- microSD card (recommended 32GB or larger)
- Camera:
  - USB webcam
- 2.5 inch HDMI touchscreen or other attached display when using kiosk mode
- Internet connection (Wi-Fi or Ethernet)
- Cooling solution such as a heatsink, fan, or active cooling case

Recommended for setup and testing:
- HDMI monitor or small HDMI display
- HDMI cable / micro HDMI cable for Raspberry Pi 5
- USB keyboard and mouse for first setup
- Raspberry Pi case with heatsink or cooling fan

Required for GPIO button trigger (Phase 6):
- Six momentary push buttons
- Default BCM mapping:
- Button Main / Capture -> GPIO17
- Button 1 / Read Text -> GPIO5
- Button 2 / Summarize Document -> GPIO6
- Button 3 / Analyze Image -> GPIO13
- Button 4 / Professional Assistant -> GPIO19
- Button 5 / Solve Problem -> GPIO26
- Jumper wires
- Breadboard (optional but recommended for easier wiring)

Optional:
- Small touchscreen / portable display
- USB speaker or buzzer for feedback in future upgrades
- Power bank / UPS for portable demo setup
