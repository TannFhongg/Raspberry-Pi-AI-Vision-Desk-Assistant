Raspberry Pi AI Vision Desk Assistant
Hardware Requirements

Required:
- Raspberry Pi 5 8GB
- Official Raspberry Pi USB-C power supply (recommended 5V/5A)
- microSD card (recommended 32GB or larger)
- Camera:
  - USB webcam
- 11.6-inch HDMI display, non-touch, used in landscape orientation
- HDMI cable or micro-HDMI-to-HDMI cable appropriate for Raspberry Pi 5
- Internet connection (Wi-Fi or Ethernet)
- Wi-Fi adapter managed by NetworkManager and supporting protected AP/hotspot mode
  (required only for phone-first provisioning; Ethernet can be used after setup)
- Cooling solution such as a heatsink, fan, or active cooling case

Recommended for setup and testing:
- USB keyboard and mouse for first setup and any non-GPIO UI navigation
- The phone-first setup portal can enter Wi-Fi and OpenAI credentials without a keyboard;
  retain the keyboard/mouse for recovery, diagnostics, and administrator-only tasks.
- Raspberry Pi case with heatsink or cooling fan

Required for non-touch GPIO operation:
- Ten momentary push buttons
- Default BCM mapping:
- Button Main / Capture -> GPIO17
- Button 1 / Read Text -> GPIO5
- Button 2 / Summarize Document -> GPIO6
- Button 3 / Analyze Image -> GPIO13
- Button 4 / Professional Assistant -> GPIO19
- Button 5 / Solve Problem -> GPIO26
- Back -> GPIO22
- Navigate Up -> GPIO23
- Navigate Down -> GPIO24
- Select / Confirm -> GPIO25
- Jumper wires
- Breadboard (optional but recommended for easier wiring)

Optional:
- USB speaker or buzzer for feedback in future upgrades
- Power bank / UPS for portable demo setup

Display configuration:
- The production QML reference resolution is 1366x768 (approximately 16:9).
- The earlier 1200x800 canvas was an incorrect display assumption and is legacy only.
- In kiosk mode, VisionDesk renders directly at the HDMI panel's actual fullscreen geometry.
- `display.size` in `config/device.yaml` and `UI_SCREEN_WIDTH` / `UI_SCREEN_HEIGHT`
  are the windowed-development size, not a command to change the monitor resolution.
- The installer adds Noto Sans and fontconfig. VisionDesk falls back to Inter,
  DejaVu Sans, or the bundled OFL Roboto face if Noto Sans is unavailable.
- This display has no touch input. The ten GPIO buttons cover normal operation; keep a
  USB keyboard and mouse available for setup and administrative text entry.
