# Upwork Project Description

Raspberry Pi AI Vision Desk Assistant is a portfolio-ready MVP inspired by a real freelance automation use case.

The project combines Raspberry Pi 5 hardware, camera capture, OpenCV preprocessing, OpenAI Vision analysis, a local Flask dashboard, and a GPIO push-button trigger. The system can capture a real-world image, clean it up for readability, send it to an AI vision model, and present the answer through either a terminal flow, a web UI, or a physical button interaction.

This project demonstrates practical skills in:

- Raspberry Pi integration
- Python backend development
- OpenCV image preprocessing
- OpenAI API integration
- Flask web UI development
- GPIO hardware interaction
- deployment and portfolio presentation
